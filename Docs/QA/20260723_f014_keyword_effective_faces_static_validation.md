日期：2026-07-23
状态：静态通过；Unity Play Mode 视觉验收待执行

## 验证范围

- 悬浮窗底部关键词标签、解释与规则正文高亮。
- `market_floor / family / dual_family` 隐藏边界。
- 喂养最低点、幸运龟最大面与六个实际生效点数。
- `9,999 / 10,000 / 99,999 / 100,000` 格式切换边界。
- 亿级科学计数、中文快速读取、唯一最高判断与 LED 字库覆盖。
- 运行时“商店 / 酒馆”旧称扫描。

| 检查 | 结果 |
|---|---|

| 隐藏解释 | 通过：`market_floor / family / dual_family` 均为 `show_explanation=0` |
| 通用说明标题与点数来源标签 | 通过：运行代码无“术语说明”和“已含喂养” |
| 龟壳类重复触发标签 | 通过：运行映射不再返回“化壳时” |
