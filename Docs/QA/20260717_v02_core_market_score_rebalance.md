## 效果语义复核

| 项目 | 当前口径 |
|---|---|
| 最高面 / 最低面 | 读取真实长期面组，允许 0 与负数 |
| 饲料最低点 | `真实最低面 + FeedValue`，负面可被逐步托高 |
| 吞吃 / 吸附转移 | 读取来源真实面组平均值，向下取整后仅对最终转移量保底为 1 |
| 幸运龟 / 双倍龟 | 幸运读取真实最大面；双倍龟把当前最小面乘 2，负面会变得更低 |
| 光牙 | 同次结算中，同一双阵营实体最多被选择一次 |
| 奇数规则 | 使用绝对值判断，负奇数可以正确触发 |
| LowFog | 只对 1 / 2 点 `-1`，不惩罚 0 与负点 |
| 双阵营身份 | 实体同时提供两个家族身份；吸附成为壳后不再提供身份 |

## 第二轮离线样本依据

- 在不启动 Unity 的前提下，按当前一次出手、结算规则、市场、购买、刷新和金币流程运行 10 个固定种子自动流程；通关关数为 `6、0、3、3、19、19、31、3、6、40`，平均 `13` 关、中位数 `6` 关，仅 1 次完整通关。
- 9 次失败分布为：`1-1` 1 次、`1-boss` 3 次、`2-3` 2 次、`5-boss` 2 次、`8-boss` 1 次，说明单关失败即结束的当前实现对首章随机低点和中期成长波动惩罚偏重。
- 六颗公平基础骰在旧首关目标 `15` 下理论通过率为 `93.92%`，即玩家尚未接触市场就有 `6.08%` 的不可控失败；新目标 `14` 将其改善为 `96.41%`。
- 只按失败检查点回放新门槛，9 个原失败点中有 6 个会被新目标接住；这不等于 6 次完整通关，因为检查点之后的随机、购买和成长路径尚未续跑。
- 母猪 + 多只三只小猪曾出现终局 `3121 / 678`，小鬼 + 龟龟成长链也出现单骰均面约 `283` 的异常高成长；这类离群值留作骰子效果上限复核，不用于抬高全局曲线。
