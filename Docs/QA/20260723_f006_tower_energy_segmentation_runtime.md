# F006 积分塔分段能量柱运行验证

日期：2026-07-23  
反馈：`F006-V7R3-FB-002`  
状态：实现与自动化运行验证通过，待用户实机体验回归

## 问题与根因

用户反馈：“积分塔结算时会出现一个长条形的光柱，但是只是一个色块处理下”。

运行时检查确认，问题来自结算表现层直接在积分塔 12 格灯体上覆盖全高 `DrawRect`：

- `BribeFinal` 使用贯穿塔内的 `6 px` 实心柱。
- Exceed / FarExceed 使用 `6 px / 10 px` 实心柱。
- Critical 蓄压和击穿使用持续塔内实心柱，并在释放点追加贯穿画面的直线色条。
- 旧回退与终局残留也保留同类整块填色。

这些矩形跨过灯格间隙，导致塔体结构和材质感消失，读成临时色块。

## 实现结果

- 新增统一的 12 格分段能量绘制，逐格对齐既有积分塔灯体。
- 每格由低透明外晕、琥珀主体、温白热芯构成，并保留灯格间隙。
- 移动继电头只在当前能量位置增加横向短火花，不形成连续长柱。
- Exceed 保留规则双拍；FarExceed 增加受降效设置约束的轻微错位；Critical 继续按原相位蓄压、真空与击穿。
- Critical 原先的贯屏直线色条改为上下分叉、带断口和支路的瞬时放电；横向冲击改为多股不同长度的热丝。
- Pass 继续使用上一轮确认的目标线短接点，不恢复全高能量柱。
- 非 Pass 终局残留改为逐灯格衰减，不再用整块矩形填满塔内。
