## 已接入行为

- `starting_lives = 3` 从 `global.csv` 读取，并以 `3` 作为配置缺失时的安全默认值。
- 生命由整轮共享，只在新游戏 `ResetRun()` 时初始化；进入新小关不会恢复。
- 每手完整结算并提交成长后，累计分仍未达标会立即扣除 `1` 点生命。
- 扣后仍有生命时保留累计分、钱包、长期面组、饲料、吸附、投资锁金和本关一次性标记，进入可重试失败表现；水平扫描完成后只清理每手状态并回到可排序的 `Ready`。
- 扣后为 `0` 时立即清除本轮存档，确认失败表现后返回开始界面，不再提供额外出手。
- 只有累计首次达标时进入既有通关收入与市场流程，失败续投不发通关收入。
- 顶部 HUD、旧版回退资源栏、失败终端和底部操作文案已显示真实剩余生命与“再次出手”语义。
