## 已记录的用户决策

- 采用竖版紧凑悬浮窗，不显示右侧“面 / 效 / 质”页签。
- 不要纸页厚度、账册装订、书脊、多层纸张或翻页感。
- 初版只显示名称、六格点面、骰子效果、品质效果和卖出价。
- 骰子效果说明来源于 `Assets/Resources/Data/dice_type_config.csv`，只用于展示。
- 停留 `0.12s` 后显示，`0.10s` 淡入，离开后 `0.08s` 淡出。
- 相邻目标切换不重新等待和重淡入。
- `Shaking / Stopping / Scoring / StageFailed` 默认不显示。
