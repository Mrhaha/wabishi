## 视觉意图

三颗龟龟补强骰应属于同一条“龟壳工坊”路线：

- 壳匠骰：修壳、工匠、数量结算、工具感。

- 慢龟骰：慢速、长链、厚壳、稳步前进。

## 必须保留

- 小尺寸角标中能区分壳匠、巢穴、慢龟。
- 图标必须以骰子为主体，玩家第一眼应先读到“这是一颗骰子”。
- 功能识别必须通过贴在骰子上的符号化配饰和骰面符号体现，不只是在骰子旁边放不同道具。
- 三颗骰子的功能动词必须第一眼不同：壳匠是“把数量转成加分”，巢穴是“补出 d1 小骰”，慢龟是“逐级慢衰生成长链”。
- 龟龟路线视觉统一，但三颗行为差异清楚。

## 避免

- 真实动物写实感。
- 幼儿贴纸或过度可爱表情。
- 图标内可读文字。
- 过多小龟细节导致市场货架不可读。
- 三颗都只是同一种龟壳骰换角度或换背景。
- 配饰大到盖过骰子，或把图标变成道具场景。
- 配饰细小到只能作为装饰，无法表达功能。
- 三颗骰只靠同类龟壳、不同挂件或不同姿态区分。
- 工具、巢托、沙漏等配饰只表达题材，不表达具体规则功能。
- 使用复杂工作台、托盘场景、沙漏场景等压过骰子主体的构图。

## 方向评审门

状态：方向 A 已接受，最终图标已生成

三颗骰属于同一条“龟壳工坊”视觉家族，但需要在图标生成前确认家族表达。方向通过后，先生成一张包含三颗骰的接触图或一颗代表样张，再决定是否批量生成最终图标。

### 方向 A：龟壳工坊

- 视觉：三颗骰都像桌面工坊里的小型功能骰，壳匠带小锤和补丁，巢穴带浅色小窝和 d1 小骰，慢龟带厚壳和缓慢轨迹。
- 轮廓：主体保持 3/4 骰子，外侧用工具、窝边和轨迹做差异。
- 玩法识别：壳匠读作“按数量修壳加分”，巢穴读作“补一颗小骰”，慢龟读作“链条走得更久”。
- 适合点：最贴合明亮账本桌游风，也最容易和现有龟龟骰形成同路线家族。
- 淘汰信号：工具太细、巢穴太像动物插画、慢龟轨迹不可读。

### 方向 B：王室小龟公文

- 视觉：三颗骰都带王室公文道具，壳匠像盖章修补许可，巢穴像备案小窝，慢龟像拖着长长审批章的厚壳骰。
- 轮廓：骰子主体更方正，配蜡封、印章、纸条和小皇冠。
- 玩法识别：用公文数量和盖章节奏暗示“数量结算、补骰、慢衰减”。
- 适合点：强化项目的荒诞官僚幽默，和账本、印章主视觉一致。
- 淘汰信号：公文元素喧宾夺主，玩家看不出龟龟路线。

### 方向 C：小龟棋子队列

- 视觉：三颗骰强调小龟路线的队列感，壳匠旁有整齐小壳片，巢穴里排出 d1 小骰，慢龟后方有逐渐变小的骰子足迹。
- 轮廓：主骰旁保留少量大块小骰剪影，避免碎细节。
- 玩法识别：最直接表达“临时小骰数量”，适合小尺寸图标和市场货架。
- 适合点：读法清楚，能服务壳匠和巢穴的数量表达。
- 淘汰信号：小骰过多导致图标拥挤，或三颗骰之间差异不足。

## 第三版修订方向：骰子主体 + 简单功能配饰

状态：未通过

总体规则：

- 每颗图标的主体必须是一颗清晰的 3/4 视角骰子，占画面视觉量约 60% 到 75%。
- 三颗骰子的主体形状保持同家族，但用不同贴身配饰区分功能。
- 配饰只保留 1 到 2 个大块元素，不做完整场景，不使用复杂背景。
- 配饰要贴在骰子上或紧邻骰子，作为类型识别，不要变成独立主物件。
- 不使用文字，不使用动物表情，不把小龟画成角色。

### 壳匠修订目标

- 主识别：骰子主体 + 贴在骰角的短柄锤 + 一块明显龟壳补片或铆钉。
- 配饰数量：锤子和补片即可，不要铁砧、工作台、夹具场景。
- 淘汰信号：只是在旁边放道具，或者工具大到盖过骰子。

### 巢穴修订目标

- 主识别：骰子主体 + 简化半环小巢托 + 一个贴边 d1 小骰。
- 配饰数量：半环托和小骰即可，不要完整草窝、托盘或自然巢。
- 淘汰信号：巢具面积超过骰子，或草窝细节密到小尺寸不可读。

### 慢龟修订目标

- 主识别：骰子主体 + 小沙漏挂坠或慢速拨针 + 一段短粗弧形轨迹。
- 配饰数量：计时符号和短轨迹即可，不要完整沙漏场景、节拍器场景或长道路。
- 淘汰信号：只靠厚龟壳表达慢，或计时器大到成为主体。

## 第四版修订方向：骰子主体 + 功能符号配饰

状态：已通过，已转最终图标

总体规则：

- 每颗图标仍以一颗 3/4 视角骰子为主体，骰子视觉量保持约 60% 到 70%。
- 功能符号必须贴在骰面、骰角、绑带或骰边，不做独立场景。
- 每颗骰子只保留一个主功能符号和一个辅助配饰，避免再次变成复杂道具画。
- 主功能符号必须能在 64 px 角标尺寸读出大意。
- 三颗骰可以共享龟壳工坊家族元素，但主功能符号的轮廓不能相似。

### 壳匠第四版目标

- 主识别：骰子主体 + 骰面或骰角上的“壳片计数牌” + 小锤或铆钉。
- 功能符号：用三片小龟壳片汇入一个加分章，表达“临时小骰数量转成壳匠加分”。
- 辅助配饰：短柄锤只作为工匠身份，不应成为唯一识别点。
- 轮廓目标：比普通龟壳骰更像“修补后的加分骰”，有明显补片、铆钉和计数感。
- 淘汰信号：只看到锤子和补片，但看不出数量转分。

### 巢穴第四版目标

- 主识别：骰子主体 + 骰边半环巢托 + 明显 d1 小骰从巢托里冒出。
- 功能符号：d1 小骰必须比第三版更大，且用单点骰面明确表达“补出 1 点临时小骰”。
- 辅助配饰：巢托只保留简化半环或护栏，不画完整自然巢、草窝或托盘。
- 轮廓目标：像“带小骰孵化槽的骰子”，不是普通骰子旁边挂一个小方块。
- 淘汰信号：小骰太小、巢托太像装饰边框，或玩家看不出补骰功能。

### 慢龟第四版目标

- 主识别：骰子主体 + 骰面上的阶梯式减半符号 + 小沙漏挂坠或短粗弧形轨迹。
- 功能符号：用从大到小的两到三颗小骰影或阶梯刻痕表达“逐级慢衰”，不要只靠沙漏表达慢。
- 辅助配饰：沙漏或轨迹只负责慢速气质，主功能必须是链条逐级变小。
- 轮廓目标：像“会拖出更长小骰链的骰子”，和巢穴的单颗 d1 补骰区分开。
- 淘汰信号：只看到慢或厚壳，看不出长链和衰减。
