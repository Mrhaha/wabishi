## 已确认默认值

- 国库骰：`floor(小关开始钱包 / 10)` 单骰加分，无单颗上限。
- 贿赂骰：每 1 金补 +4 最终分；每颗每次最多 2 金；只有能补到通关时才花钱。
- 投资骰：`min(投资骰数量 * 2, floor(小关开始钱包 / 3))` 自动锁定预算；每颗最多 2 金；每金让本骰每次结算 +2 单骰分。
- 市场投放：国库 `T2 / 10 / 5 / 6-9-7`，贿赂 `T2 / 11 / 5 / 5-8-8`，投资 `T3 / 12 / 5 / 2-5-7`。
- 存档：新类型使用类型 key，旧类型序号仅作为兼容输入。
