### F006-V7R3-FB-005 修改方案与回归标准

#### 体验翻译

- 玩家要的不是“更亮、更大、更多射线”，而是清晰的受力反差：整台机柜先像人下蹲一样被压力压低并停住，再从积分塔内一个很小的热核向上顶穿。
- 现有运行时击穿同时向左右、上下和全屏扩张，释放阶段持续绘制整机负载与托盘扫光；视觉能量足，但主方向更像闸门打开后的洪水。
- 本轮保持已确认的寂静窗口与 `2.20s` 总时长，不重新加快爆炸；改的是爆发前后的位移曲线、能量集中度和释放方向。

#### 修改边界

| 相位 | 新动作 | 通过标准 |
|---|---|---|
| 下压 | 真空前机柜内容持续向下沉，顶部 / 底部压力夹向积分塔核心 | 不看文案也能感到机器先“蹲下”，不是只变暗 |
| 锁死 | 横向流动停止，周边能量收束到积分塔中央小热核 | 击穿前至少有一小段“压住不动”的硬停顿 |
| 顶穿 | 爆心先形成窄而快的向上喷柱，再打开上方扇形冲击 | 第一眼主方向是向上顶开，而不是左右同时泄放 |
| 过冲 | 机柜内容在击穿后明显上冲至顶点 | 动作与下压方向相反，形成“蹲下—跳起” |
| 回落 | 顶点后回落并以一次较小反弹进入余震 | 不在峰值后持续平推；回落能被感觉到但不拖长总时长 |

#### 根因
