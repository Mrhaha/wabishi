# ACCEPTANCE

状态：已确认

## 纸面验收

- 只有 1 个主体家族 + 接济：获得 1 金币。
- 4 个主体家族 + 接济：获得 4 金币。
- 缺少 3 个主体家族 + 空袭：空袭自身 +6。
- 没有主体家族 + 空袭：不触发，不获得 +8。
- 4 个主体家族 + 缝合：缝合自身 +8。
- 盟约左右两侧同家族：不触发。
- 盟约左右两侧不同家族：左右邻居各 +2。
- 光牙在四族阵容中：每个主体家族各 1 颗骰子获得所有骰面 +2。
- 光牙成长不回改本次锁定有效点；保存恢复后永久骰面成长保留。
- 二重唱左侧为可复制家族效果：该效果再次触发 1 次。
- 二重唱左侧为中立、市场、购买、卖出、刷新、离场、纯被动或复制效果：不复制。
- 触发器遇到 3 次真实主体家族结算事件：对应的 3 个事件源各获得临时基础分 +2；若同一源触发多次则可重复获得。
- 触发器遇到一个影响多个目标的群体效果：只计 1 次事件。
- 二重唱复制事件：不触发触发器。
- 王冠在三家族阵容中：分别读取三个家族最高最终有效点，获得向下取整均值作为额外单骰分。
- 王冠在无主体家族阵容中：不获得额外分。
- 可吸附龟作为实体骰时计为龟龟家族；完成吸附后不再计为龟龟家族，宿主只按自身类型计家族。

## 原型验收

- 8 颗中立骰子能通过测试入口出现。
- 8 颗中立骰子的 tooltip 与实际行为一致。
- 中立骰子不会破坏恶魔、龟龟、猪猪、海盗的既有结算。
- 二重唱和触发器不存在无限递归。
- 中立效果不会被二重唱复制，也不会触发触发器。
- 光牙随机目标在日志中可见。
- 王冠回合结束触发顺序稳定。

## 体验验收

- 玩家能把接济、空袭、缝合理解为过渡骰。
- 玩家能把光牙、二重唱、触发器理解为跨家族放大器。
- 玩家能把王冠理解为中立终局核心。
- 混搭阵容在前中期不会因为缺少完整家族而完全断战力。
- 纯中立阵容不应成为最简单的终局答案。
- 玩家能理解吸附龟成为壳层后不再提供龟龟家族数量。

## 文档验收

- `DICE_ARCHETYPES.md` 记录中立家族定位和 8 颗骰子名单。
- `GAME_FLOW.md` 记录中立骰子涉及的结算阶段和回合结束阶段。
- `PROJECT_CONTEXT.md` 记录中立家族仍处于 V0.2 测试池，不是正式第五家族。
- 若实现行为和本包默认值不同，必须回写本生产包。
