# ART_BRIEF

状态：原型美术就绪，正式美术后置  
阶段：原型期可使用占位图；后续进入正式美术生产前需要单独出方向稿。

## 视觉方向

中立家族建议走“议会 / 王冠 / 盟约 / 拼合”的跨家族视觉，不做成第五个强烈阵营。它应该像连接器，让玩家直觉上知道这些骰子在读取、组合、复制其他家族。

## 风格关键词

- 明亮、清晰、桌游感。
- 金属、牙光、缎带、徽章、拼接纹、王冠轮廓。
- 不要过暗，不要做成恶魔支线。
- 不要让海盗、龟龟、猪猪、恶魔的符号被中立图标抢占。

## 单骰图标方向

- 光牙骰子：发光獠牙或白金牙形徽记，表示成长咬合。
- 二重唱骰子：双音符、双面谱线或成对声波。
- 触发器骰子：按钮、扳机或机关芯，但避免真实枪械感。
- 王冠骰子：小王冠与四枚家族小徽记环绕。
- 接济骰子：补给袋、援助手、金币包。
- 空袭骰子：从空位落下的光标或降落标记，避免军事轰炸感过重。
- 盟约骰子：双侧握手、契约绶带或左右徽章连接。
- 缝合骰子：拼布骰面、缝线和多色碎片。

## 交付物

原型阶段：

- 8 个 256x256 占位图标。
- 1 个中立家族临时徽记。

正式阶段：

- 8 个正式骰子图标。
- 1 个中立家族徽记。
- 结算小特效参考图：光牙成长、二重唱复制、王冠读取。

## 验收标准

- 小尺寸下能区分 8 颗中立骰子。
- 不与四个主体家族图标混淆。
- 王冠和盟约要有明显“跨家族”语义。
- 接济、空袭、缝合要能看出是过渡补位，而不是终局主核。
