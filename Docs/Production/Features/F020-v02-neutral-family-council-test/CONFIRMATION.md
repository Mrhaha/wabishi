# F020 确认记录

状态：已确认

来源设计对话：D1  
关联审查：`EXECUTION_REVIEW.md`

## 已确认范围

- 中立骰子首批名单保持为：光牙、二重唱、触发器、王冠、接济、空袭、盟约、缝合。
- 中立定位为四个主体家族之间的过渡、兼容与终局收束，不作为第五个独立闭环家族。
- 四个主体家族固定为猪猪、恶魔、龟龟、海盗；中立不计入主体家族数量。

## 用户裁决

1. 可吸附龟在吸附成为壳层后不再计为龟龟家族，只读取被吸附宿主自身的家族；吸附壳不会让宿主获得第二家族身份。
2. 二重唱和触发器只处理主体家族的真实结算事件，市场、购买、卖出、刷新、离场、中立事件、纯被动和二重唱复制事件均排除。
3. 光牙的所有面 +2 为永久成长，写入骰面状态，但不回头改变本次已经锁定的有效点数。
4. 王冠读取每个已有主体家族本回合最终有效点最高的一颗骰，取向下取整均值，并把该均值作为王冠的额外单骰分。

## 同步采用的审核默认值

- 触发器按每次真实可见主体家族结算事件计次，不按效果键全局去重；一个群体效果同时影响多个目标仍算一个事件。
- 触发器把本次临时基础分 +2 给事件源骰，不改变有效点数或骰面；二重唱复制事件不触发触发器。
- 二重唱在自身槽位结算时复制左邻本次产生的最后一个允许复制的主体家族结算事件一次。
- 空袭至少需要存在一个主体家族才触发；纯中立阵容不获得缺族加分。
- 盟约要求左右两侧都为主体家族且家族不同；左右邻居各获得本次临时单骰分 +2，盟约自身不加分。
- 缝合按已有主体家族数获得本次临时单骰分 +2 / 家族，最高 +8。
- 接济按已有主体家族数获得金币，进入钱包流。
