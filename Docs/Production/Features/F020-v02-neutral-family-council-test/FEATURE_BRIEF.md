# F020 V0.2 中立家族议会王冠测试

状态：已确认  
来源：`Docs/DesignDialogues/D1-future-dice-types.md`，2026-07-14 用户完成规则裁决  
父级方向：V0.2 四家族主体逻辑完成后，补充中立 / 跨家族过渡骰子  
目标版本：V0.2 原型测试

## 一句话目标

加入一组“可过渡、可混搭、终局最难成型”的中立骰子，让玩家在恶魔、龟龟、猪猪、海盗之间切换时有稳定桥梁，并让最终形态依赖各家族核心强骰共同完成。

## 设计边界

- 中立家族不是第五个普通家族，不应该单靠纯中立池完成低难度通关。
- 中立骰子要在前中期有实际战力或金币价值，避免只在终局才有意义。
- 中立终局需要吃到多家族最强骰子的收益，因此成型难度应高于单家族阵容。
- 不给四族主线增加新的硬上限；只在复制、监听、随机成长等可能造成无限连锁的地方设置触发边界。
- 先进入 V0.2 测试池，不立即承诺最终数值。
- 主体家族只统计猪猪、恶魔、龟龟和海盗，中立自身不计入。
- 可吸附龟成为宿主壳层后不再计为龟龟家族，宿主只保留自身家族；首领龟等仍作为实体存在的龟龟骰正常计为龟龟家族。
- 本包的基础分、自身加分和邻居加分均为本次临时单骰分，不改变有效点数、骰面或家族判定，除非规则明确写为永久所有面成长。

## 核心骰子名单

### 光牙骰子

基础意图：多家族成长核心。  
用户规则：每有一个不同家族，结算时，各个家族随机一个骰子获得各个骰面 +2。  
已确认执行值：统计当前实体骰组中的主体家族；每个已有主体家族随机选择 1 颗该家族实体骰子，使其所有骰面永久 +2。成长写入面组和存档，但不回头改变本次已锁定有效点数。同一实体每次光牙结算最多获得一次光牙成长。

### 二重唱骰子

基础意图：复制左侧家族效果，成为跨家族连锁支点。  
用户规则：左边骰子触发效果时，可以再次触发一次。  
已确认执行值：结算到二重唱时，复制左侧相邻骰子本次产生的最后一个允许复制的主体家族结算事件 1 次。不复制中立、市场、购买、卖出、刷新、离场、纯被动、二重唱复制事件或二重唱自身效果。

### 触发器骰子

基础意图：监听其他主体家族效果，给实际触发者补战力。  
用户规则：每触发一次其他骰子效果，使对应骰子获得基础分 +2。  
已确认执行值：每发生 1 次真实可见的主体家族结算事件，事件源骰本次临时基础分 +2。一个群体效果同时影响多个目标仍算一次；不按效果键全局去重。中立、市场、购买、卖出、刷新、离场、纯被动和二重唱复制事件不计入。

### 王冠骰子

基础意图：中立终局收束，读取各家族最强骰子的结果。  
用户规则：回合结束，获得其他各个家族的各一个骰子的点数均值。  
已确认执行值：回合结束时，从每个已有主体家族中选取本回合最终有效点最高的 1 颗实体骰，王冠获得这些点数的向下取整均值作为额外单骰分。不复制效果，不改变被读取骰子；没有主体家族时额外分为 0。

### 接济骰子

基础意图：低风险经济过渡。  
用户规则：每有一个不同家族，结算金币 +1。  
已确认执行值：结算到接济骰子时，按当前实体骰组中的主体家族数获得金币；金币进入本回合钱包流，可被后续依赖金币的效果读取。

### 空袭骰子

基础意图：缺族补位，降低过渡阵容的战力断档。  
用户规则：每缺少一个家族骰子，自身 +2。  
已确认执行值：至少存在 1 个主体家族时，以四个主体家族为基准，每缺少 1 个主体家族，自身本次临时单骰分 +2。没有主体家族时不触发，避免纯中立阵容依靠空袭独立成型。

### 盟约骰子

基础意图：位置型双邻居增益，鼓励混搭排布。  
用户规则：左右两侧不同骰子，左右 +2。  
已确认执行值：左右相邻格都存在主体家族实体骰，且两侧主体家族不同，则左右邻居各获得本次临时单骰分 +2；盟约自身不加分。中立骰和吸附壳不参与家族差异判断。

### 缝合骰子

基础意图：最朴素的多家族战力过渡。  
用户规则：根据已有的不同家族数 +分。  
已确认执行值：按当前实体骰组中的主体家族数，自身本次临时单骰分 +2 / 家族。最高四族时 +8。

## 成型路径

前期：接济、空袭、缝合提供金币和基础分，允许玩家先混入任意家族过关。  
中期：盟约开始要求位置经营，光牙开始为多家族核心骰堆永久面值。  
后期：二重唱和触发器围绕家族强效果放大收益。  
终局：王冠读取各家族高点数结果，形成“多家族最强骰 + 中立收束”的最高难度阵容。

## 非目标

- 不在本包重做四个主体家族。
- 不新增圣遗物或市场系统大改。
- 不把中立设计为无脑经济流。
- 不允许二重唱或触发器读取市场、离场、中立、纯被动或复制事件。

## 执行切片

1. F020-01 中立测试池与基础数据：加入 8 颗中立骰子的配置、文案、临时图标占位和市场测试开关。
2. F020-02 家族识别与结算事件：统一实体主体家族数、缺失家族数、结构化主体家族结算事件和最终有效点读取等基础接口。
3. F020-03 过渡战力骰：实现接济、空袭、盟约、缝合。
4. F020-04 连锁成长核心：实现光牙、二重唱、触发器，并加入防递归边界。
5. F020-05 王冠终局：实现回合结束读取各家族点数均值，并验证最高难度成型路径。
6. F020-06 表现、提示和验收：补齐结算浮字、骰子提示、纸面试玩记录、文档同步。

## 已确认执行口径

- 光牙永久改骰面，不回改本次锁定点数。
- 二重唱只复制左邻最后一个允许复制的主体家族结算事件一次。
- 触发器按真实主体家族结算事件计次，不按效果键去重；奖励事件源骰临时基础分 +2。
- 王冠读取各主体家族最高最终有效点，均值向下取整后作为额外单骰分。
- 吸附龟成为壳层后不再计为龟龟家族，只读取宿主自身家族。
- 空袭在没有主体家族时不触发；盟约只读取两侧不同主体家族。
