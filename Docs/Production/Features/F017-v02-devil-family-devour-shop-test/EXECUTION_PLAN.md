## 已确认默认值

- 吞吃值：吞吃和后续吸附共用同一套期望转移逻辑；首版整数面组使用当前六面平均值向下取整，最低 1。
- 小鬼：提高本轮持续市场基础最低点。
- 魔蝠：每颗魔蝠离场时最多吞吃 1 颗可用市场骰。
- 市场补货：购买后货架留空，不自动补货；只有进入市场和付费刷新会整批补货。
- 深渊召唤：至少一颗时，进入市场和付费刷新固定 1 个贡品，多颗不叠加。
- 测试模式：新增恶魔单家族市场测试开关，首测只随机刷五颗常规恶魔骰，贡品只由深渊召唤强制生成。
