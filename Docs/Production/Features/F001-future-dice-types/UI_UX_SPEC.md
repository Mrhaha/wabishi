## 体验意图

F001 的新增骰 UI 目标是让玩家在市场、未投掷、结果预览、出千和结算动画中都能用短标签理解路线收益。每颗新骰都不能依赖长段说明。

## F001-D 界面重点

| 路线 | 市场标签方向 | 结果标签方向 | 预览行方向 | 结算反馈方向 | 状态 |
|---|---|---|---|---|---|
| 大树命中 | 命中、差一点、照料 | 命中点、灌溉命中 | 大树是否成长 | 成长和面组变化 | 待规则确认 |
| 大树成熟 | 年轮、树冠、成长层数 | `年轮 +N`、`树冠 +N` | 成长兑现收益 | 单骰加分 | 待规则确认 |
| 新树培养 | 苗圃、新树 +1 | 市场买入提示 | 不常驻结算 | 买入时反馈 | 待规则确认 |

## 边界情况

- 出千后大树命中点不能刷新，但预览必须按新骰面重算。
- 灌溉如果直接视为命中，必须和自然命中区分。
- 苗圃如果只影响未来买入大树，市场提示不能误导为已有大树也会成长。
