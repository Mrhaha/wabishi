- 在 `DEV_HANDOFF.md` 中明确首版运行时重载最小实现：默认读 `Resources`，热调至少支持 `Application.persistentDataPath/roll_feedback_config.csv` 覆盖；启动或重载日志需要显示外部覆盖路径；`StreamingAssets` 作为可选，不作为首版硬依赖。

## 风险

| 风险 | 影响 | 缓解方式 | 负责人 |
|---|---|---|---|
| 热重载来源不明确 | 程序可能只实现 `Resources`，用户无法运行时修改配置 | 明确外部覆盖文件路径和日志 | 程序 |
| 随机隔离验收不可执行 | 没有确定性种子时无法证明敲击不影响结果 | 增加随机消耗代码审查和临时调试验证 | 程序 |
| 音效字段误解为必做 | 无音效资源时阻塞实现 | 标记音效字段首版可解析但可不接入 | 美术 / 程序 |
| 旧文案遗漏 | 玩家仍看到“延迟停止”，破坏新规则理解 | 在程序交接列出旧文案接入点 | 界面体验 / 程序 |
| 重载中途改变当前投骰 | 动画跳变或结果误解 | 当前投骰配置快照不可变，新配置下一次投骰生效 | 程序 |
