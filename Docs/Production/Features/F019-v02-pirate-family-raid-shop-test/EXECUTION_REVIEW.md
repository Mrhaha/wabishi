## 摘要

F019 不再按“保守经济流”处理，而是按“市场操盘 + 赃金转战力”执行。海盗头领的折价和可能出现的短期套利不是缺陷，而是海盗家族的爽点之一。真正需要防止的是真无限、队列卡死和不可读，所以程序实现应优先保证刷新价格递增、三格货架、购买后货架处理、六槽容量和市场结束这些自然边界成立。

同时，买入价和卖出价需要沉淀为基础通用规则：二者都从同一套骰子估值公式派生，不再把海盗头领写成孤立价格特例。
