## 功能验收清单

### 测试市场

- [ ] 开启 `v02_pirate_family_market_only` 后，市场只出现 8 颗海盗骰。
- [ ] 关闭测试开关后，不影响当前其它测试池或正式市场路径。

### 统一价格公式

- [ ] 买入价和卖出价都从同一套骰子估值公式派生。
- [ ] 成长面组、负面组、随机售价和真实吸附回收能影响对应估值或实例售价。
- [ ] 海盗头领只影响海盗骰买入价，不直接改卖出估值。
- [ ] 允许短期折价或倒卖收益；循环应被刷新价格递增、三格货架、货架置空、六槽容量和市场结束自然阻止。

### 刷新海盗

- [ ] 0 颗刷新海盗时，刷新费用保持原逻辑。
- [ ] 1 颗刷新海盗时，刷新净费用减少 1。
- [ ] 多颗刷新海盗时，减费可累加。
- [ ] 刷新净费用为 0 时可免费刷新。
- [ ] 刷新净费用为负时按差额返还金币，并触发财宝海盗金币事件。
- [ ] 刷新基价每次刷新后仍正常递增。

### 劫掠海盗

- [ ] 左右相邻实体骰全部面永久 `-1`。
- [ ] 面组可以变成 0 或负数。
- [ ] 负点数在结算中直接扣分，不被错误夹到 0。
- [ ] 劫掠金币进入钱包，并能被后续财宝海盗读取。
- [ ] 保存恢复后负面组仍存在。

### 人口海盗

- [ ] 主动卖出 2 颗骰后生成 1 颗基础骰。
- [ ] 生成基础骰面组为 `1 / 2 / 3 / 4 / 5 / 6`。
- [ ] 生成基础骰具有随机售价。
- [ ] 本市场多次卖出可以多次触发。
- [ ] 骰袋满时不生成，并显示停止原因。
- [ ] 随机售价保存恢复正确。

### 海盗头领

- [ ] 购买海盗骰时买价半价。
- [ ] 非海盗骰不受影响。
- [ ] 买价、卖价和刷新组合可以产生短期折价收益，但不能形成不消耗货架、不受刷新递增约束的稳定无限循环。

### 练腿海盗

- [ ] 买入任意骰后，随机其它己方骰全部面永久 `+1`。
- [ ] 没有其它骰时不触发。
- [ ] 成长在悬浮窗和保存恢复中可见。

### 财宝海盗

- [ ] 每个真实金币入账事件后，随机己方骰全部面永久 `+1`。
- [ ] 不按金币数量逐枚触发，而按入账事件触发。
- [ ] 不设置次数上限。
- [ ] 不因自身成长产生新的金币事件。

### 抢劫海盗

- [ ] 点击离开市场后，若有抢劫海盗，市场操作锁定。
- [ ] 抢劫海盗从非空货架随机获得 1 颗骰。
- [ ] 被抢货架置空且不自动补货。
- [ ] 新骰加入骰袋最后。
- [ ] 新骰若有离场效果，加入队列末尾继续触发。
- [ ] 抢到抢劫海盗后可以继续递归。
- [ ] 货架空时自然停止。
- [ ] 骰袋满时自然停止。
- [ ] 队列清空时进入下一小关。
- [ ] 不出现同一货架被无限重复抢取。

### 海贼王

- [ ] 金币 0 到 2 时成长 0 次。
- [ ] 金币 3 到 5 时自身全部面永久 `+1`。
- [ ] 金币 6 到 8 时自身全部面永久 `+2`。
- [ ] 高金币时无上限成长。
- [ ] 成长不消耗金币。
- [ ] 保存恢复后成长面组正确。

### 界面与反馈

- [ ] 刷新按钮能显示刷新净费用和海盗减费。
- [ ] 负面组能在悬浮窗和结算中清楚显示。
- [ ] 抢劫队列能显示当前触发、获得对象、新位置和停止原因。
- [ ] 海贼王悬浮窗能显示当前金币对应成长次数。
