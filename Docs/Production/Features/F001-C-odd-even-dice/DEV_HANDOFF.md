## 代码影响

当前实现把奇偶二版效果拆成三个清晰阶段：

1. 结果生成后：自然开盅结果锁定时处理孤证骰自动重摇；出千确认后的强制结果不调用孤证检查。
2. 手牌识别时：半步骰提供顺子候选值。每颗半步骰最多提供原值或低一格借位值中的一个选择，多颗半步骰可以各自参与同一条顺子。
3. 单骰计分时：盖章骰和轨道骰先替换本骰基础单骰分，再按有效点数应用遭遇规则修正。

需要避免把半步骰写成全局顺子规则。它应是“本次有半步骰时，顺子识别可尝试该骰自己的借位值”，且不能让同一颗半步骰同时贡献两个值。

固定分顺序说明：

- 当前实现沿用计分管线：类型效果先改本骰分，遭遇规则再按有效点数修正。
- 例：轨道骰触发后基础单骰分为 8；若遭遇规则让偶数骰 `+1`，最终单骰分可变为 9。
