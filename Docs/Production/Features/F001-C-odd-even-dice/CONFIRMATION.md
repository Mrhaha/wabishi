## 方案甲说明

盖章骰或轨道骰触发后，先把本骰基础单骰分替换为固定分；随后遭遇规则仍按有效点数修正这个单骰分。

示例：轨道骰触发后基础单骰分为 8；如果当前遭遇规则让偶数骰 `+1`，最终单骰分可变为 9。
