## 建议改进

### 已写回生产包补丁

1. 孤证骰首版只在自然开盅后的结果锁定触发，不在出千确认后的强制结果中再次触发。
2. 半步骰允许多颗同时参与同一条顺子判定，但每颗半步骰在一次判定中只能选择原值或低一格借位其中之一；它只服务顺子，不服务三同、全奇、全偶或其它牌型。
3. 半步骰实现建议使用小规模候选值搜索：每颗半步骰提供两个候选值，普通骰提供一个候选值，判断是否存在一组“一骰一值”的选择形成 5 连或更长顺子。
4. 轨道骰固定分数采用 8 分，且只读取最终实体骰有效点数；临时小骰不参与，多颗轨道骰在 `2 / 4 / 6` 齐备时可同时触发。
5. 盖章骰固定分数采用 6 分。
6. 固定分采用方案甲：固定分先作为本骰基础单骰分，之后仍受遭遇规则修正。
