## 风格关键词

- 明亮账本桌游风。
- 羊皮账本、印章、王冠、记分牌、桌面道具。
- 数字清楚、形状厚实、对比明确。
- 低文字、低装饰噪音、清晰层级。
- 轻松、机敏、桌游感，不幼态。

## 必须保留

- `骰子王` 标题需要在首屏可见。
- 关卡信息要保留，但视觉权重低于中央计分器。
- 左侧资源只服务金币、出手和出千。
- 中央骰子和双数字计分器是主视觉。
- `0 / 72` 不要被装饰盖住。

## 避免

- 避免把“当前骰袋”“构成”“全部上场”“本关关注”等被删除信息重新画成 UI 区块。
- 避免给基础分和倍数加大段说明文字。
- 避免暗色赌场、霓虹、写实 3D 或过度金属质感。
- 避免过多边框、徽章、丝带同时抢数字焦点。
- 避免在接触图里生成难以替换的中文小字；运行时文案应由 Unity 绘制。

## G 版确认视觉语言

- 左侧资源牌：金币、出手、出千使用等宽等高小账牌，图标在左，数字在右，保持统一中心线。
