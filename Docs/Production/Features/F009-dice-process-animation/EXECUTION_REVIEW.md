## 风险

| 风险 | 影响 | 缓解方式 | 负责人 |
|---|---|---|---|
| OnGUI 序列帧坐标不稳定 | 骰子可能漂出投掷区或遮挡左侧积分 | 只在投掷区内绘制六骰，并保留 2D 回退 | 程序 / 界面体验 |
| 动画结果与真实骰面不同步 | 玩家误解结果来源，破坏可信度 | 停转定格只读取已确定结果；动画层不得自行随机最终面 | 程序 |
| 结算点名重复触发计分 | 钱包、成长或临时小骰可能重复应用 | 点名只读取 `SettlementDisplayEvent`、`scoringDice` 和 `RunScoreCounterState`；不得调用 `ScoreDice()` | 程序 |
| 动画拖慢高频投骰 | 重复游玩节奏变差 | 普通投骰停转显点维持约 1.2 到 1.8 秒，点名每骰约 0.15 到 0.25 秒 | 界面体验 / 程序 |
| 临时程序骰丢失类型身份 | 玩家看不懂非基础骰和状态标签 | 首版至少保留类型角标、色条或图标叠层，最终资源后置 | 美术 / 界面体验 |
