## 建议默认值

- 首版采用 OnGUI 序列帧播放器。
- 首版不新增最终美术资源。
- 首版不新增音效、震动或加速结算。
- 首版先覆盖自然投骰，出千重摇后置。
- 普通投骰停转显点目标约 1.2 到 1.8 秒。
- 结算点名每骰约 0.15 到 0.25 秒。

## 验证计划

- 静态检查：`git diff --check`。
- 结构检查：动画层不调用 `ScoreDice()`。
- 运行检查：普通投骰、出千前后、三同 / 顺子倍率、金币触发、龟龟临时小骰、大树成长、贿赂补分。
- 回退检查：关闭 F009 视觉层后原流程仍可玩。
- 视觉检查：`1280x720` 和 `1920x1080` 录屏 / 截图。
- 文档检查：实现后同步 `PROJECT_CONTEXT.md` 和 `GAME_FLOW.md`。
