## 建议默认值

- 首版采用 OnGUI 序列帧播放器。
- 首版不新增最终美术资源。
- 首版不新增音效、震动或加速结算。
- 首版先覆盖自然投骰，出千重摇后置。
- 普通投骰停转显点目标约 1.2 到 1.8 秒。
- 结算点名每骰约 0.15 到 0.25 秒。
