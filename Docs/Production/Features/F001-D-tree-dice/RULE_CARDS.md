# F001-D 规则卡

状态：已实现待验证
最后更新：2026-06-10

## 园丁骰

定位：大树自然命中后的爆发加速。

规则：

1. 自身正常按有效点数计分。
2. 每棵自然命中的大树获得基础成长 1 次。
3. 每有 1 颗园丁骰，该自然命中的大树额外成长 1 次。
4. 多园丁叠加，多棵自然命中的大树分别吃到全部园丁。

边界：

- 只吃大树自己的自然命中。
- 灌溉补成长不触发园丁。
- 不改变点数、牌型或分数公式。

## 灌溉骰

定位：大树未自然命中时的补成长。

规则：

1. 自身正常按有效点数计分。
2. 有效点数等于未自然命中大树的命中点数时，让该大树成长 1 次。
3. 每颗灌溉每次最多浇中 1 棵大树。
4. 同一棵大树每次最多获得 1 次灌溉成长。

边界：

- 灌溉成长不算自然命中。
- 不触发园丁。
- 出千重摇灌溉骰面后按新结果重新判断。
