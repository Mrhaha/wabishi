# F001-D 界面体验规格

状态：已实现待验证
最后更新：2026-06-10

## 展示目标

- 不新增长说明面板。
- 继续使用短标签、市场短建议和结算日志表达机制。

## 当前标签

- 园丁骰未投掷：`大树加速`
- 园丁骰投掷后：有自然命中大树时显示 `园丁 +N`，否则显示 `等树命中`
- 灌溉骰未投掷：`浇中成长`
- 灌溉骰投掷后：命中可浇大树时显示 `灌溉 N` 或 `灌溉命中`，否则显示 `等命中点`
- 大树自然命中：继续显示 `大树命中`
- 大树被园丁放大：追加 `园丁 +N`
- 大树被灌溉：追加 `灌溉成长`

## 日志

- 园丁触发时记录自然命中大树数量和每棵额外成长次数。
- 灌溉触发时记录浇中的未自然命中大树数量。
- 大树应用成长时按最终成长次数汇总，例如 `成长 3 次`。
