## 已确认执行口径

- 吸附扫描发生在离开市场进入下一小关前。
- 双倍龟在吸附时一次性翻倍宿主当前最小面。
- 呼朋唤友龟按自身宿主的结算位置触发；若它先结算，本次后续骰子会计算该效果。
- 首领龟不吸附到其它骰子上，是独立实体骰。
- 多颗首领龟可以同时响应同一真实吸附事件，但首领之间、复制壳之间不会互相触发。
- 首领复制壳和基础骰子壳不计入真实吸附卖价。
- 幸运龟只影响有效结算值，不算自然最大面。
- 双倍龟遇到多个并列最低面时全部翻倍。
