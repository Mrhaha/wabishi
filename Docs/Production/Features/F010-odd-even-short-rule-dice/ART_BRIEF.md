七颗骰都属于奇偶短规则族，应共享奇偶、槽位、复核、出千印章等视觉语言；同时每颗必须通过骰体结构或骰面嵌件表达不同机制。玩家不读文字时，也应能初步区分邻位、补全、复核、翻号、守号和转号。

## 接触图单元

| 骰子 | 需要表达的机制 | 建议符号方向 | 避免 |
|---|---|---|---|
| 异邻骰 | 左邻奇偶不同 | 骰体左侧异色邻位槽、奇偶双色嵌件 | 三槽夹心构图 |
| 同邻骰 | 左邻奇偶相同 | 骰体左侧同色邻位槽、成对同类封章 | 与异邻只靠颜色深浅区别 |
| 补全骰 | 补全全奇 / 全偶 | 缺口被补齐的奇偶环、合拢封印 | 画成长或金币 |
| 复核骰 | 破坏全奇 / 全偶时重摇 | 内嵌回环刻线、复核章 | 大型外部箭头 |
| 翻号骰 | 出千后奇偶改变给分 | 双色翻面印、奇偶翻转刻槽 | 与转号骰完全同构 |
| 守号骰 | 出千后奇偶没变给分 | 同色锁扣、稳定封签 | 表现成禁止出千 |
| 转号骰 | 出千重摇必变奇偶 | 奇偶分界转盘、强制换色机关 | 表现成直接加分 |

## 风格关键词

- 明亮账本桌游风。
- 2D 扁平、清晰轮廓、浅纸质感、软阴影。
- 奇偶双色、封章、刻槽、骰面嵌件、桌游零件感。
- 轻松、机智、不幼态。

## 必须保留

- 骰子本体是第一视觉主体。
- 点数孔或骰面结构仍然可读。
- 接触图不含可读中文或英文文字。
- 机制符号尽量内嵌在骰面、骰边、骰角或骰体结构中。
- 七颗在小尺寸下仍要能区分。

## 避免

- 避免赌场霓虹、写实 3D、金属豪华风。
- 避免幼态吉祥物、表情脸、贴纸感。
- 避免大外部箭头、大牌匾、大说明道具。
- 避免把 `异邻 / 同邻 / 翻号 / 守号 / 转号` 做成只有颜色不同。
