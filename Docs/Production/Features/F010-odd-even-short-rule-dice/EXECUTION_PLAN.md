## 执行顺序

1. 使用 `$wabish-packet-review` 审查 F010 规则卡、默认边界和接触图验收标准。
2. 使用 `$wabish-art-production` 生成七颗接触图，输出到 `Assets/ArtSource/Production/F010/`。
3. 用户评审已生成的接触图：接受、修订或暂停。
4. 接触图接受后，再生成最终透明 PNG 并登记 `ART_ASSETS.md`。
5. 使用 `$wabish-dev-implementation` 进入程序实现。
6. 执行样例验证、截图验收和主文档同步。
