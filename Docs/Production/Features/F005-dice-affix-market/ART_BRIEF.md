## 输入决策

- 首批只需要三类改造道具：添加词缀、删除词缀、替换并新增词缀。
- 道具命名采用项目内官署和工坊气质：`加印石 / 剥印石 / 换印石`。
- 美术只表现道具类别，不需要为每条词缀做独立图标。
- 风格必须继承 `Bright Ledger Boardgame` / `明亮账本桌游风`，避免直接照搬其它游戏的宝石或球体。

## 视觉意图

三类道具应让玩家一眼理解“给骰子盖印、剥掉旧印、换掉旧印”的行为。它们是商店里的小型工具或印章材料，不是大型装备，不应抢过骰子商品本身。

### 方向 A：官署石印组（已确认）

- 视觉：三枚小型石印或印章材料，配蜡封、盖印痕、剥落旧印和换印双层结构。
- 轮廓：加印石是完整石印加新封，剥印石是裂开的旧封和刮片，换印石是双面转印或新旧叠印。
- 配色：加印偏青绿和朱红，剥印偏灰白和褪红，换印偏蓝紫和金黄。
- 玩法识别：最贴近“词缀像印章盖在骰子上”的机制隐喻。
- 适合点：官署、账本、印章气质强，和现有项目风格一致。
- 淘汰信号：如果看起来像普通宝石、法术水晶或稀有材料，则方向失败。

### 方向 B：工坊印模工具组

- 视觉：三件小型工坊工具，分别是盖印模、剥印小凿、换印夹具。
- 轮廓：工具感更强，剪影靠把手、凿尖、夹具开合区分。
- 配色：暖木色、铜边、浅石色，少量朱红印泥提示。
- 玩法识别：强调“改造骰子”的工坊操作感。
- 适合点：适合商店道具栏，小尺寸下动作方向清楚。
- 淘汰信号：如果过于像武器、铁匠装备或维修工具，弱化印章概念，则方向失败。

### 方向 C：账本封签材料组

- 视觉：封签、票据、印泥和小石片组合成三类消耗品。
- 轮廓：加印是新封签，剥印是撕开的旧封签，换印是两张交叠替换的封签。
- 配色：羊皮纸、朱红、墨蓝和少量金色。
- 玩法识别：强调商店和账本手续，弱化工具感。
- 适合点：和 UI 账本风最贴合，适合日志与市场卡片。
- 淘汰信号：如果小尺寸下像普通纸条或任务票据，不能直接读出添加、删除、替换，则方向失败。

推荐先试：方向 A。它最贴合“词缀被盖到骰子上”的核心表达，也最不容易和现有骰子类型图标混淆。

## 风格关键词

- 明亮账本桌游风。
- 小型石印、蜡封、官署印章、刻痕、标签、骰面符号。
- 清楚外轮廓、低复杂度、小尺寸可读。
- 道具之间颜色和轮廓要能区分，但不要形成强烈霓虹或稀有度炫光。

## 必须保留

- 明亮、干净、轻松但不幼态。
- 道具应能放入商店卡片小图区域。
- 图标中不要出现可读文字，最终文案由 Unity UI 绘制。
- 图标不应让玩家误以为是新的骰子类型。

## 避免

- 赌场筹码、宝石球、法术水晶、暗黑装备材料。
- 复杂背景、满屏粒子、强稀有度光效。
- 直接在图里生成中文或英文文字。
- 把道具画成大型角色或吉祥物。
