# BG3-G 九十市场三策略轨迹与供给控制审计

审计日期：2026-07-23  
对应设计对话：`D2` 第 `54` 轮  
审计对象：第 `52` 轮九十个固定市场、`BG3-F` 五条严格导演资格、保守 / 成长 / 破局三种规则代理  
状态：纸面轨迹审计完成；`BG3-G` 供给控制参数仍为候选，未进入代码、配置或正式玩法文档

## 一、结论

`BG3-F` 可以继续作为导演资格底稿，暂时没有证据要求修改十二张法术的价格。

- 将严格资格回灌九十个固定市场后，共移除或重分类 `5` 条旧法术方向，但 `90 / 90` 个市场仍保有至少两条可辩护方向。
- 不需要新增法术纠偏、骰位替换或第二颗骰位替换；第 `52` 轮的 `19` 次法术纠偏、`9` 次单骰位替换和 `81 / 90` 不替换骰位结论不变。
- 三种策略共形成 `270` 条第一承诺轨迹。`79 / 90` 个市场至少出现两种不同首选，其中 `14 / 90` 三种策略各走不同首选；只有 `11 / 90` 三者收敛到同一首选。
- 法术架未满的 `243` 条策略轨迹中，立即施放 `57`、跨店保存 `3`、明确跳过 `183`。单张法术第一选择率最高的是饲料袋 `59.0%`，没有超过 `60%`；所有法术的明确跳过率都高于 `25%`。
- 这个结果不能证明跨店保存已经平衡。九十个样本是互相独立的单市场切片，没有给出下一市场货架、钱包和法术架演化；只有 `3 / 243 = 1.2%` 能在不虚构未来的前提下判为明确保存。保存行为必须用连续市场状态另测。
- 法术位当前的主要问题不是普遍过强，而是晚期即时可执行率不足：早期 `23 / 27 = 85.2%`、中期 `14 / 27 = 51.9%`、晚期 `9 / 27 = 33.3%`。货架施肥、黑市分账和换班凭证在本批自然样本中都没有成为第一选择。

因此形成候选 `BG3-G`：保留卡价和三段基础权重，给法术位增加“可执行 / 可明确保存 / 纯诱惑”三档供给倍率、连续空转保护和严格的五项预测计数。它只控制功能空转，不保证家族核心或指定卡名。

## 二、`BG3-F` 回灌结果

### 1. 被移除或重分类的五条旧方向

| 市场 | 旧方向 | 严格判定 | 回灌后结果 |
|---|---|---|---|
| `L1-03` | 买血契船长后换班 | 普通卖弱资产、装入新骰已由基础托盘免费完成，换班被支配 | 改记为“买血契船长进托盘，卖弱资产后装入”；仍为两方向 |
| `L4-03` | 买恶魔骰子后换班 | 普通换代没有托盘专属处理或首次装入顺序收益 | 改记为普通托盘采购、卖旧、装入；仍为两方向 |
| `L6-03` | 买三只小猪后换班 | 换班没有产生基础流程之外的新结果 | 改记为普通托盘采购、卖旧、装入；仍为两方向 |
| `L2-02` | 施肥后采购 / 吞吃 | 公开状态只能证明一件商品会被处理，不能证明至少两件 | 删除施肥方向；其余三方向保留 |
| `L3-01` | 分账后吞货架 | “可吞货架”没有给出预计吞吃数，不能证明 `n >= 3` | 删除导演计数；其余四方向保留 |

前三条不是市场可操作性下降，而是把本来属于两格交易托盘的自由度错误记到了换班凭证名下。这个修正强化了托盘价值，也避免高阶法术用六金重复基础操作。

### 2. 继续成立的严格方向

| 法术 | 市场 | 成立原因 |
|---|---|---|
| 金币饲料 | `M6-03`、`M9-03`、`L8-02` | 钱包可完成至少四次真实喂养，不只使用最低两次档 |
| 金币饲料 | `L4-02` | 高饲料质量、肉猪、贪吃与驮粮形成六次喂养容量 |
| 清仓祭典 | `L7-02` | 三格非空、至少两个有效目标，并有清仓猪 / 金币读取 / 长期成长后续 |

### 3. 导演必须新增的显式计数

不能再从模糊标签推断法术方向。市场生成日志至少要提供：

- 黑市分账：`predicted_market_devours`、金币事件响应数、是否打开下一现金阈值。
- 金币饲料：计划喂养次数、饲料质量、肉猪 / 贪吃 / 驮粮等响应层数。
- 清仓祭典：非空货架数、有效目标数、成长 / 现金 / 下一刷三类后续。
- 换班凭证：回收后可执行的托盘专属动作、首次装入顺序是否改变结果。
- 货架施肥：本市场预计会被采购、化壳或吞吃的不同商品实例数。

缺少计数时按“不足以替导演补方向”处理，但普通随机法术位仍可以低权重展示为诱惑或跨店投资。

## 三、三种规则代理

三种代理先执行同一组硬门槛，再按不同优先级选择第一承诺动作。

共同硬门槛：

1. 整条采购、施法、追加支付和后续处理可支付。
2. 法术通过 `BG3-F` 资格；只可保存的法术不替导演补当前方向。
3. 托盘、法术架、目标与实例数量合法。
4. 普通卖旧装新、同一资产同一用途和纯按钮顺序不重复计为方向。
5. 跨店保存必须由当前持久资产或确定流程保证下一阶段用途；“下一店也许有目标”一律记为跳过。

| 策略 | 首要目标 | 可接受代价 | 典型首选 | 主动停止点 |
|---|---|---|---|---|
| 保守 | 当关稳定、现金与利息档 | 不接受一次掉两档利息；只有止损链允许牺牲长期面组 | 低价整刷、即战件、可负担性法术、卖托盘 | 完成最低安全改造，或下一动作将破坏保留预算 |
| 成长 | 永久面组、点火、宿主与长期引擎 | 可掉一至两档利息，只要永久资源入口明确 | 点火骰、化壳、饲料、拆解、混合成长 | 完成一个可持续成长节点，避免为同一节点继续堆投入 |
| 破局 | 动作链长度、市场资源回流与阈值打开 | 可牺牲利息和边际资产，但链必须有自然停止 | 刷新经济、吞吃、材料处理、清算、招募、清仓 | 输入、目标或托盘材料耗尽，或下一次刷新边际转负 |

同分时依次选择：当前结果可预览更完整、不可逆损失更小、金币支出更低、货架位置更靠左。规则代理不是战力最优求解器，只用于检查同一公开状态是否会因目标不同产生可复现分叉。

## 五、`BG3-G` 供给控制候选

### 1. 不改价格，先给法术位三档供给倍率

每次生成法术位时先按原三段权重抽取候选，再根据公开、可复现的行动资格乘以下倍率：

| 档位 | 定义 | 候选倍率 | 能否替导演补方向 |
|---|---|---:|---|
| 可执行 | 本市场可完成整条立即链，且不被基础动作支配 | `1.00` | 可以 |
| 可明确保存 | 当前不能高效施放，但持久资产与确定流程已保证下一阶段用途 | `0.70` | 不可以 |
| 纯诱惑 | 可购买，但没有完整当前链，也没有确定保存用途 | `0.35` | 不可以 |
| 不合法 | 无合法目标、架满或支付链不成立 | `0` 参与纠偏；普通展示按原规则记录但不可购买 | 不可以 |

倍率只控制法术功能空转，不读取“玩家缺哪张核心骰”，也不把指定家族牌塞给玩家。固定种子必须记录基础权重、资格档、倍率和最终抽取。

### 2. 连续空转保护

- 若连续两个法术位都属于“纯诱惑”，且玩家均明确跳过，第三个法术位只从不同主功能且至少为“可明确保存”的候选中抽取。
- 保护只执行一次；之后恢复正常权重，不冻结商品、不锁定具体卡名。
- 同名、最近三次同名、同主功能和玩家近期拒绝仍使用 `BG3-E` 的记忆衰减。
- 目标不是让每张法术都能立刻用。候选观察区间为：法术架未满时整体立即可执行率 `50%–70%`，晚期不低于 `45%`，同时保留至少 `25%` 的明确跳过。

### 3. 五张严格牌的资格倍率

| 法术 | `1.00` 条件 | `0.70` 条件 | `0.35` 条件 |
|---|---|---|---|
| 货架施肥 | 本市场至少处理两件商品 | 持久引擎能确定在下一市场处理两件 | 只能抬数、当前最多处理一件 |
| 黑市分账 | `n >= 3`，或两次加金币响应 / 阈值后续 | 吞吃引擎持久存在，下一市场三份食物可确定 | 最多一至两次且无后续 |
| 金币饲料 | 至少四次容量，或高质量多响应 | 成熟饲料路线存在但当前目标 / 质量时机不对 | 只需要最低两次且无响应 |
| 清仓祭典 | 三格、两目标和完整后续 | 持久清仓引擎存在，下一市场三格确定 | 两格以下或销毁关键采购机会 |
| 换班凭证 | 回收后立即接化壳、拆解、组合体处理或独特首次装入 | 特殊资产处理窗口已确定在下一市场出现 | 只是普通卖旧装新 |

### 4. 本轮建议

- 保留十二张法术当前价格。
- 保留三段基础权重，暂不把零首选直接解释为弱牌。
- 接受 `BG3-F` 五条导演资格，并新增上述显式预测字段。
- 以 `BG3-G` 三档倍率和连续空转保护进入下一轮连续市场纸测；倍率未通过连续状态前不写入正式配置。
