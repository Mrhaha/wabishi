# BG3-H 三市场连续序列与存牌资格审计

审计日期：2026-07-23  
对应设计对话：`D2` 第 `55` 轮  
审计对象：第 `52` 轮三十个资源切片组成的连续三市场序列、保守 / 成长 / 破局三种规则代理、`BG3-G` 开启 / 关闭对照  
状态：纸面连续轨迹审计完成；`BG3-H` 仍为候选，未进入代码、配置、存档版本或正式玩法文档

## 一、结论

`BG3-G` 的三档供给倍率可以保留为下一轮候选底稿，但不能原样进入生产。它解决了“法术位经常纯空转”的问题，同时暴露出“可明确保存”资格过宽。

- 主样本为 `30` 个连续三市场状态 × `3` 种策略，共 `90` 条序列、`270` 个连续市场决策；另以相同输入关闭 `BG3-G` 运行 `90` 条对照序列，不计入主样本。
- 法术架未满时，自然抽取到“当前可执行”法术的比例由关闭时的 `92 / 253 = 36.4%` 提高到开启时的 `142 / 252 = 56.3%`，进入 `50%–70%` 观察区间。
- 开启后的自然分段为早期 `58.1%`、中期 `53.6%`、晚期 `57.3%`；晚期高于 `45%` 下限。
- 经过原有硬导演的最多一次法术纠偏与一次非保底骰位纠偏后，最终可执行率为 `172 / 252 = 68.3%`。这个最终率不用于约束供给倍率的 `70%` 上限；供给上限只看自然层，硬纠偏率单独记录。
- 硬导演介入没有增加：法术纠偏由 `39` 次降到 `30` 次，骰位纠偏由 `7` 次降到 `6` 次，完全不干预的市场由 `224 / 270` 提高到 `234 / 270 = 86.7%`。
- 整刷没有被法术搜索替代。关闭 / 开启时完整整刷动作分别为 `175 / 176` 次，基本不变；总有效动作由 `499` 增至 `526`，说明新增的是“法术与采购之间的操作链”，不是把整刷按钮抹掉。
- 开启后总利息收入由 `217` 降到 `207`，即九十条序列合计少 `10` 金、每条序列少约 `0.11` 金。当前属于玩家主动选择付出的轻微机会成本，不需要系统补偿。
- 开启后共购买保存 `39` 张法术，其中 `27` 张发生在第一或第二市场，真正于剩余一至两个市场内施放的只有 `5 / 27 = 18.5%`。其中原本被标为“可明确保存”的 `14` 张只有 `1` 张兑现，说明当前保存资格存在大量假阳性。
- 法术架满状态出现 `18` 次，实际阻止购买 `11` 次，但规则代理没有一次愿意丢弃旧牌换新牌。因此本轮既不能证明“零金丢弃”有必要，也不能证明它不必要；需要专门的架压样本。
- 当前不改十二张法术价格。除只出现一次的换班凭证外，所有出现至少三次的法术都保有不低于 `30.8%` 的明确跳过率；换班凭证样本不足，不能据一次施放判断过强。

因此形成候选 `BG3-H`：保留 `1.00 / 0.70 / 0.35` 供给倍率与单次空转保护，收紧保存资格，把自然供给与硬纠偏分开验收，并禁止导演替玩家补偿其主动把利息花空后的资源不足。

## 二、连续序列口径

### 1. 样本结构

- 将第 `52` 轮的 `E1-01 → E1-02 → E1-03`、`E2-01 → E2-02 → E2-03` 依次组成十条早期序列。
- 中期同理组成 `M1–M10` 十条序列，晚期组成 `L1–L10` 十条序列。
- 每条序列分别运行保守、成长、破局三种策略代理。
- 同一套种子与策略再关闭三档倍率和空转保护，形成镜像对照。

这些 `-01 / -02 / -03` 原本是三个固定市场样本，本轮把它们按编号重组为连续状态。它适合验证资源携带和规则自洽，不等于真实随机局的出现概率，也不替代真人策略。

### 2. 跨店携带

每次离店后携带：

- 当前钱包。
- 法术架与已存法术。
- 已装入的点火、成长、吞吃、化壳、刷新经济等持久状态。
- 同名、同功能、最近纠偏与最近拒绝记忆。
- 托盘处理结果；托盘必须在离店前清空，不跨市场保存实体。
- 当前市场离店时的下一次刷新价格，只作为该店日志保存。

跨店收入继续遵守现行经济：离店钱包先获得固定 `+3` 金，再按收入后的钱包每 `6` 金获得 `1` 金利息，最多 `4` 金。

同一市场内付费刷新价格递增；进入下一市场后重置到章节基础刷新价。只有刷新海盗等持久减费状态跨店携带，不能把上一店已经递增的数值错误带进下一店。

### 3. 三种策略代理

| 策略 | 主要目标 | 允许牺牲 | 本轮新增的连续状态判断 |
|---|---|---|---|
| 保守 | 当关下限、现金、利息 | 尽量不掉两档利息 | 会为了保住下一店购买力保存便宜的确定牌 |
| 成长 | 永久成长、点火、宿主 | 可掉一至两档利息 | 只在持续资产能兑现时存牌，不为纯搜索保险占架 |
| 破局 | 动作链、吞吃、刷新、阈值 | 可主动花到零利息 | 会保留高上限法术，但必须有自然停止条件 |

代理不是战力最优解，也不代表真人购买率。它只用于检查三种可复现偏好在同一供给规则下是否仍能形成不同资源轨迹。

## 三、`BG3-G` 开关对照

### 1. 自然供给与硬导演

| 指标 | 关闭 `BG3-G` | 开启 `BG3-G` | 变化 |
|---|---:|---:|---:|
| 法术架未满样本 | `253` | `252` | `-1` |
| 自然可执行 | `92 / 253`（`36.4%`） | `142 / 252`（`56.3%`） | `+19.9pp` |
| 硬纠偏后可执行 | `131 / 253`（`51.8%`） | `172 / 252`（`68.3%`） | `+16.5pp` |
| 法术纠偏 | `39` | `30` | `-9` |
| 非保底骰位纠偏 | `7` | `6` | `-1` |
| 完全不干预市场 | `224 / 270` | `234 / 270` | `+10` |
| 连续空转保护触发 | `0` | `6` | `+6` |
| 方向底线失败 | `0` | `0` | 不变 |

自然供给按章节分段：

| 阶段 | 关闭自然可执行 | 开启自然可执行 | 开启后硬纠偏结果 |
|---|---:|---:|---:|
| 早期 | `37 / 87`（`42.5%`） | `50 / 86`（`58.1%`） | `56 / 86`（`65.1%`） |
| 中期 | `28 / 84`（`33.3%`） | `45 / 84`（`53.6%`） | `57 / 84`（`67.9%`） |
| 晚期 | `27 / 82`（`32.9%`） | `47 / 82`（`57.3%`） | `59 / 82`（`72.0%`） |

晚期硬纠偏后的 `72.0%` 不表示供给过密。`50%–70%` 是自然法术位的观察区间；硬导演只在市场不足方向底线时工作，验收的是干预频率与是否暗送答案。

### 2. 经济与操作

| 指标 | 关闭 `BG3-G` | 开启 `BG3-G` | 判断 |
|---|---:|---:|---|
| 完整整刷动作 | `175` | `176` | 法术没有替代整刷经济 |
| 总有效动作 | `499` | `526` | `+27`，约 `+5.4%` |
| 累计利息收入 | `217` | `207` | 玩家为更多操作合计让出 `10` 金 |
| 本市场立即施法 | `71` | `87` | 可执行供给增加 |
| 购买保存 | `32` | `39` | 表面增加，但兑现质量不足 |
| 后续施放存牌 | `5` | `5` | 保存增加没有转化成更多兑现 |
| 法术架阻塞购买 | `11` | `11` | 供给倍率未恶化阻塞 |

这个结果符合目标体感：法术给玩家多一层资源用途，但整刷仍然是搜索和刷新经济的核心动作。

### 3. 主动耗尽不由导演免费修复

连续状态中有 `57` 个开启样本是在玩家前一店主动投入后，以零利息档进入下一市场。若仍硬保“两条完整战略方向”，导演会被迫暗送便宜牌或免费价值，相当于替破局策略还债。

`BG3-H` 建议区分：

- 进入市场仍有至少一档利息时，继续要求至少两条后果不同的可辩护方向。
- 玩家在前店主动花到零利息、并以零利息档进入后续市场时，只保证至少一个当前可支付、合法的动作，不保证系统把第二条路线免费补回来。
- 仍禁止真正死店：至少要能整刷、买入、施放、处理托盘或用已有资产完成一项有结果的操作。

这不是降低“每店有事可做”的标准，而是区分随机供给把玩家锁死与玩家自己选择透支资源。

## 四、三种策略结果

| 策略 | 架未满样本 | 最终可执行 | 立即施法 | 保存 | 后续施放存牌 | 跳过 | 整刷 | 有效动作 | 利息 | 序列末钱包均值 |
|---|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| 保守 | `83` | `61`（`73.5%`） | `34` | `14` | `2` | `36` | `69` | `184` | `91` | `10.20` |
| 成长 | `84` | `60`（`71.4%`） | `27` | `7` | `2` | `52` | `51` | `169` | `64` | `6.23` |
| 破局 | `85` | `51`（`60.0%`） | `26` | `18` | `1` | `45` | `56` | `173` | `52` | `2.97` |

三种策略没有被供给控制压成同一种玩法：

- 保守策略通过利息与低风险整刷保留最多后续购买力。
- 成长策略把更多金币换成长实体，因此末钱包处于中段。
- 破局策略最常为了引擎与长链主动压低利息，末钱包显著最低。

## 五、存牌资格需要重写

### 1. 兑现率

- 开启 `BG3-G` 后共购买保存 `39` 张。
- 其中 `27` 张发生在第一或第二市场，理论上仍有一至两个市场的兑现窗口。
- 只有 `5 / 27 = 18.5%` 在窗口内真正施放。
- `13` 张是“当前可执行但策略主动延后”，其中 `4` 张兑现。
- `14` 张是供给分类器认定的“可明确保存”，其中只有 `1` 张兑现。

问题不在三格法术架本身，而在“有持久资产，未来也许能用”被误判成了确定保存。例如：

- 已有成长路线，不等于下一店必然有值得拆解的材料。
- 已有某家族，不等于下一店定向招募的额外购买成本仍可支付。
- 已有宿主，不等于下一店必然出现第二个值得外壳转运的目标。
- 下一店刷新较贵，不等于现在保存单格换货一定优于保金币。

### 2. `BG3-H` 保存资格

法术只有同时满足下列四项，才进入 `0.70` 的“可明确保存”档：

1. **持久输入明确**：写出已经存在且不会在离店时消失的输入、宿主、引擎或目标。
2. **兑现窗口明确**：能指出不超过后续两个市场的可预测使用窗口；“以后总会有用”不成立。
3. **未来成本可支付**：购买法术后，按下一次固定收入与利息计算，仍能支付施法及其必需的追加采购 / 刷新成本。
4. **不被持币支配**：与不买法术、保留金币相比，保存必须改变目标、顺序、资源用途或退出点。

缺少任一项，降为 `0.35` 的纯诱惑。玩家仍可主动购买纯诱惑牌，档位只影响供给权重，不禁止高风险下注。

## 六、候选 `BG3-H`

1. 保留基础权重和 `可执行 1.00 / 可明确保存 0.70 / 纯诱惑 0.35`。
2. 保留连续两次纯诱惑且均跳过后的单次不同功能保护；本轮只触发 `6` 次，没有形成持续追发。
3. 自然法术位按整体 `50%–70%`、晚期不低于 `45%` 验收；硬导演输出不参与自然上限，改看法术纠偏、骰位纠偏和未干预率。
4. 有利息档进入市场时保两条方向；因玩家主动花空而零利息进入后续市场时，只保一个付费合法动作，不免费补偿第二条路线。
5. 硬导演仍最多纠偏一张法术、再纠偏一颗非高阶保底骰；不替换第二颗骰，不保证指定家族或卡名。
6. “可明确保存”必须通过持久输入、两店内窗口、未来全成本可支付、不被持币支配四项证明。
7. 三格法术架继续保留为候选。零金丢弃既不确认也不否决，等待专门架压矩阵。
8. 十二张法术暂不改价；先修分类器，再看严格连续样本。
