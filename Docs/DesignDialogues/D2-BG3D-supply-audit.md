审计日期：2026-07-23

## 一、结论

这版结构可以继续推进，不需要为了“每店有事可做”立即重写现有骰子基础权重。

- 共审计 `90` 个固定市场：第一至二章、第三至五章、第六至十章各 `30` 个；每个资源切片包含 `2` 个按当前权重生成的基线市场和 `1` 个从合法池中寻找的压力市场。
- 原始结果已有至少两个可辩护方向的市场为 `68 / 90 = 75.6%`，高于“至少 `40%` 完全不纠偏”的透明度底线。
- 导演使用法术位纠偏 `19 / 90`，替换骰位 `9 / 90 = 10.0%`；`81 / 90 = 90.0%` 的市场没有替换骰位，高于“至少 `70%` 不替换骰位”的底线。
- 纠偏分类为：完全不纠偏 `68`、只改法术 `13`、只改一个骰位 `3`、法术与一个骰位都改 `6`。没有市场需要第二个骰位替换。
- 纠偏后 `90 / 90` 个市场均达到至少两个不同首个承诺动作；高阶保底骰没有被替换。
- 基线市场原生通过 `55 / 60 = 91.7%`；刻意寻找坏组合的压力市场原生通过 `13 / 30 = 43.3%`。压力样本用于找结构漏洞，不能反推正式游玩中的坏市场自然出现率。

真正需要调整的不是卡池价格，而是导演判定器：

1. 只有“买牌、施放、支付后续成本、执行下一动作”的完整链条可成立时，法术才能替导演填入第二方向。
2. `单格换货` 在早期整刷只需 `1–2` 金且三格都不可用时，不能被算成独立解法；它只有在保留至少一件有用商品，或当前整刷价格高于 `2` 金时才具备不同机会成本。
3. `加急清算` 只有资产现金回收值至少为 `5`，并且额外现金确实打开新动作或解决资产退出时，才能被导演计数。
4. 法术纠偏必须执行同名、同功能与近期拒绝记忆衰减。没有衰减时，早期纠偏中的 `饲料袋` 会占 `7 / 10`；加入候选衰减后降到 `6 / 10`，刚好触及而不超过 `60%` 观察线。

## 三、可辩护方向判定

本次不把按钮数量、动画次数或“以后也许有用”计作方向。一个方向必须满足：

- 首个承诺动作不同：购买某功能件、出售 / 处理托盘物、付费刷新、施放某张法术等；同一资产的同一用途换一种按钮文字仍只算一个方向。
- 当前可支付：采购与法术要计算整条行动链的总支出，不能只看第一步按钮是否亮起。
- 当前有用途：购买转化件必须已经有输入、能点火或能作为明确材料；纯粹塞进骰袋等未来配件不算导演当前方向。
- 付费刷新只有在支付后仍保守地留有 `6` 金常规采购能力、刷新免费 / 净返还，或当前吞市场引擎让刷新本身产生资源时才计数。随机刷到低面折价商品不是保证，因此不按偶然的 `3` 金最低显示价放宽。
- 法术可以被玩家买来跨市场保存，但“只可保存”不能被导演拿来宣称本店已有第二方向。
- `定向招募` 必须在支付 `5` 金后仍能买下所生成家族候选，或已有吞市场等立即消费者；仅有合法家族目标不够。
- `拆解许可` 允许“先买入托盘，再支付 `3` 金拆解”的完整链，因此不要求进店时托盘已经有材料。
- `清仓祭典` 自身就是吞吃入口，不要求预先拥有另一条吞吃入口；但必须有至少两格可吞商品，并有成长、响应或资源转换价值。
- 主动离店和保利息仍是玩家真实选择，但不作为导演给任意市场补齐第二方向的万能垫数。

导演记忆候选值为：上一次纠偏出现同名法术乘 `0.15`，最近三次纠偏内出现同名乘 `0.40`，与上一次纠偏同主功能再乘 `0.50`；只有一个完整行动链候选时允许绕过衰减。应用后纠偏法术分布为：

- 第一至二章：饲料袋 `6 / 10`、瑕疵折价 `3 / 10`、单格换货 `1 / 10`。
- 第三至五章：金币饲料 `2 / 7`，饲料袋、加急清算、定向招募、单格换货、瑕疵折价各 `1 / 7`。
- 第六至十章：单格换货、饲料袋各 `1 / 2`。

这只是“导演介入时展示了什么”的结构统计，不是玩家第一选择率。单张法术首选不超过 `60%`、合法可支付法术至少 `25%` 被跳过，以及局部搜索令整刷下降不超过 `50%`，仍必须由带选择策略的模拟或实机轨迹验证。

## 七、关键反例与卡牌判断

### 1. 合法目标不等于完整方向

`E6-02` 是最清楚的假阳性：玩家有 `7` 金，货架有 `6` 金刷新海盗，法术位是 `5` 金定向招募。只检查“能买法术且能选家族”会得到两个方向；但施放后只剩 `2` 金，无法买下新生成的常规候选，也没有吞市场消费者。严格判定只有“买刷新海盗”一条，导演必须纠偏。

在全部 `90` 局中，共有 `26` 次法术虽然可以购买或有形式合法目标，却不能立刻完成有用后续。它们可以作为玩家主动储存投资，但不得作为导演的第二方向。

### 2. 单格换货不能取代整刷

第一至二章整刷通常只需 `1–2` 金。若三格都不可用，花 `2` 金只换一格同时保留两个坏格，严格弱于更便宜或同价的整刷；此时单格换货不能用于纠偏。

它在两类局面成立：

- 至少已有一件想保留的商品，需要只修掉最差一格，例如 `E3-02`、`E7-02`、`E8-01`。
- 当前整刷已涨到 `3+` 金，需要用 `2` 金局部搜索节省搜索预算，例如 `M9-01`、`L3-03`。

这不修改卡牌合法性，只修改导演是否能把它计作“独立、非支配方向”。

### 3. 加急清算不是低值资产保险

设资产正常现金回收值为 `r`。支付 `4` 金后，加急清算获得 `r + min(r, 6)`；与直接出售相比，它的额外净现金为：

`min(r, 6) - 4`

因此：

| 回收值 `r` | 相比正常出售的净变化 | 审计判断 |
|---:|---:|---|
| 1 | `-3` | 明确拒绝 |
| 2 | `-2` | 明确拒绝 |
| 3 | `-1` | 明确拒绝 |
| 4 | `0` | 现金持平但失去出售响应，通常拒绝 |
| 5 | `+1` | 只有打开新动作时成立 |
| 6+ | `+2` | 高值资产换当期购买力 |

`E4-01`、`E7-01`、`M2-02`、`M6-01` 都展示了低回收值或已有更好路径时合理跳过；`M3-01` 展示了高值资产清算的成立场景。当前建议保留卡面与 `4` 金价格，但将导演资格锁为 `r >= 5` 且确实打开采购、刷新或必要退出，不把它当通用坏市场保险。

### 4. 五张高风险法术都有拒绝理由，但行为比例尚未验证

| 法术 | 成立场景 | 合理拒绝场景 | 纸面结论 |
|---|---|---|---|
| 瑕疵折价 | `E1-01` 用总计 `8` 金取得原价 `10` 的短期接济件 | `E2-01` 的低价龟与猪本已可买，长期面组各 `-1` 可能比省钱更差；`E3-03` 的三个商品都没有当前用途 | 保留；必须提供购买前六面预览 |
| 加急清算 | `M3-01` 用回收值 `6` 的资产换当期高钱包与退出权 | 回收值 `1–4`，或虽为 `5+` 但没有打开新动作时 | 保留；增加导演资格门槛 |
| 金币饲料 | `L4-02` 的高质量饲料与成熟猪龟能吸收六次喂养 | `L6-01`、`L7-01` 没有合适饲料入口或高容量收益，十金总支出让位于采购 / 刷新 | 保留；高质量数值仍需专项上限测试 |
| 清仓祭典 | `L7-02` 有清仓猪、金币读取与三格食物，吞货架、采购、刷新互相竞争 | 若三件都是准备购买的高价值组件且没有吞吃响应，支付 `8` 金销毁购买机会应明确劣于采购或保留 | 卡面可继续；随机 `90` 局只自然出现一次，必须补定向对照与净资产测试 |
| 换班凭证 | `L1-03` 满袋、`16` 金，买 `10` 金血契船长后支付 `6` 金原子换班 | 新骰只是边际侧换、旧骰为高成长资产，支付 `6` 金并承担旧资产清盘会劣于保持构筑或刷新 | 结构成立；需要实体龟、组合宿主和恢复存档专项用例 |

上表证明“存在可拒绝理由”，不等于已经证明每张牌在至少 `25%` 合法样本中会被真人跳过。随机审计对清仓祭典与换班凭证的自然出现数也不足；这两项仍是进入实现前的定向卡牌审计。

### 5. 拆解许可需要显示整条购买链

`E5-01`、`E5-03`、`M2-03` 等样本证明，拆解许可不应只在托盘已有材料时亮起。玩家可以先买一颗便宜骰进入托盘，再施放法术完成转化。界面需要同时预览：

- 当前金币能否支付“目标骰买价 + 法术 `3` 金”；
- 哪个托盘格接收材料；
- 材料转移值、最终六面变化；
- 被拆解骰不会获得卖价，龟不会化壳。

## 八、推荐的导演调整版

保留 `BG3-D` 的“一次最多改一张法术加一个非保底骰位”上限，执行顺序调整为：

1. 按当前骰子章节权重、价格模板、高阶保底和独立法术权重生成原始市场。
2. 用完整行动链与首个承诺签名统计方向；少于两条才进入纠偏。
3. 从能形成新签名的法术中，按“章节基础权重 × 同名记忆 × 同功能记忆 × 玩家近期拒绝记忆”抽取。候选必须先通过完整链，记忆只改变合法候选的相对概率。
4. 法术纠偏后重新计算；仍不足两条时，只替换一个最低可用、非高阶保底骰位。
5. 替换骰只按缺失功能桶寻找当前章节合法候选，不读取当前家族去补指定核心。
6. 若一个法术和一个骰位仍不能达到两条方向，记录为卡池 / 价格失败并回改权重；绝不继续替换第二颗骰。

首版建议记录下列透明日志：

- 原始三骰、原始法术、显示价格与价格模板；
- 原始方向签名与不足原因；
- 法术纠偏候选、被记忆衰减的候选及最终选择；
- 是否替换骰位、被保护的高阶保底位；
- 玩家最终首个承诺动作、跳过的合法法术、整刷次数和主动停止原因。

## 九、未通过或未覆盖的验收项

这轮只通过了供给结构下限，没有宣称卡表已经可以直接生产：

- 未测试真人或选择代理的第一选择率，因此“单张法术不长期超过 `60%` 首选”和“合法法术至少 `25%` 被跳过”仍未通过。
- 未对比加入单格换货 / 定向招募前后的整刷次数，因此“常规整刷下降不超过 `50%`”仍未通过。
- 未计算分数曲线、每金成长、利息机会成本和关卡通过率；`瑕疵折价` 的六面损失与 `金币饲料` 的高质量上限仍需数值审计。
- 未跑黑市分账加清仓祭典的一至三实例净资产与重复状态检测；这是上线阻断项。
- 未跑换班凭证的实体龟、组合宿主、首次装入、退出重进和存档恢复；这是上线阻断项。
- 压力样本会主动偏向死法术与死转化件，不能用其中的卡名出现次数估算正式卡池频率。
