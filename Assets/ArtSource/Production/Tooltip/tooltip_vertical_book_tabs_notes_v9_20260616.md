# 骰子悬浮窗 UI V9：竖向册页页签

状态：待用户确认的接触图规格，不是运行时资源。

## 本轮设计修正

- V8 的页签偏扁平贴片，仍像贴在卡片边缘的小标签。
- V9 改为竖向册页结构：悬浮窗整体更高更窄，读起来像一本小账册。
- 页签从右侧纸页后伸出，带错层阴影和纸页厚度。
- 左侧增加书脊和铆钉感，强化竖版账册实体感。

## 字段保持不变

- 头部：骰体小图、骰子名称、卖出价格。
- 中段：六格点面。
- 下段：骰子效果一行、品质效果一行。
- 不新增统计、边界标签、价格规则、推荐语或大骰预览。

## 组件规格倾向

- 主体建议宽约 `280`，高约 `360`。
- 右侧页签额外宽约 `48`。
- 页签文本使用竖向两字结构，例如“点面”“骰效”“质效”。
- 商店待购骰和主投掷区骰共用同一竖版悬浮窗。
- 越界时整本卡向左或向上翻转，保持页签和纸页错层一致。
