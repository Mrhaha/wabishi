## R4 直接吸收语法

1. **无常驻入口**：积分塔空闲时只有原有玻璃、灯格与机柜结构，不存在方框、孔洞、夹片或额外容器。
2. **末段压缩**：积分包在最后约 `18%` 路程内收窄横向体积并回正，抵达玻璃时成为短促竖向薄光，而不是钻入一个洞口。
3. **玻璃接触**：命中窗口只出现无边框局部光晕、玻璃切光和压缩线；三者随命中能量出现，静止时完全不可见。
4. **塔内接管**：接触后生成一条短寿命水平灯带，自底向上穿过塔内灯格；灯格承击波和压力记忆继续表达贡献大小。
5. **暴击归塔**：最终欠压与击穿仍由塔内部完成，接触点不会在没有积分包命中时单独发光，也不会被误读为暴击按钮。
6. **声音去夹口化**：删除每次命中的电磁夹口咬合，替换为较轻的玻璃切光与磷光吸收音；塔内继电器、目标双扣和暴击重击继续保留。

## 评审重点

- 空闲与两次命中之间，积分塔右侧是否完全没有“等待投入口”的静态物件感。
- 光包末段是否明确读成撞上玻璃并被塔吸收，而不是进入插槽、孔洞或传送门。
- 上行灯波是否把反馈继续交给积分塔内部，同时没有变成独立弹道或长光束。
- 移除夹口后，每次积分贡献的重量差仍是否足够清楚。
