## 目标

- 六颗骰子全程固定原槽，不位移、不缩放、不互相连线。
- 每个真实即时贡献者只产生一包离散积分灯珠，直接进入左侧本关积分塔。
- 主来源、回声来源和重贡献共用一条连续压力包络，不拆成三段独立停顿。
- 不显示逐骰 `+N`、百分比、过关档位名或暴击文字；强弱只通过贡献包密度、塔内灯排负载、继电器反应和声音层数表达。
- 最终高潮只释放一次，并限制在积分塔局部：欠压、继电器吸合、灯组过驱、顶部目标灯锁定、局部 CRT 同步撕裂与少量触点火花。

## 动态评审方式

- 主判断使用原速 `60fps` 有声 MP4，包含全部 `90` 张源帧。
- 需要在聊天内直接看动态图时使用 `60fps` 全帧 WebP，同样包含全部 `90` 张源帧，无声音。
- 需要看清短促追灯、欠压和继电器高潮时使用 `0.5×` 慢放 MP4，`60fps / 180` 帧，时长 `3.00s`。

- 九帧图只用于定位关键节点，不用于判断连续性。
