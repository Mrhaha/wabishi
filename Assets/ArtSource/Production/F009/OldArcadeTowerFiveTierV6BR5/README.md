## 目标

在不改变 V6B-R4 完整结算时间轴、玻璃直吸收语法和注意力分配的前提下，派生“未达到、过关、超过、远超过、暴击”五档隐藏结果反馈。玩家不通过档位名、百分比或逐骰大数字判断强弱，而通过积分塔的闭合程度、灯格材质、回弹次数、余辉、机柜负载和声音冲击自然感知本次出手层级。

## 共同冻结合同

- 总长 `4.80s`；一次 `Space` 后完成固定骰体面内滚轴、左到右停靠、自动交接、六槽主结算、局部回环与结果驻留。
- 六颗骰子始终原槽；实际贡献者只在自身骰面析出一枚小型积分灯珠包，直接接触左侧积分塔玻璃。
- 积分塔没有常驻入口、插槽、孔洞、夹片、完整命中环或第二交互物件；接触后立即由塔内上行灯带接管。
- 回环事件仍留在原主槽的局部队列中：第二槽保持主聚焦，第一槽短促回声，并从第一槽自身再次送出贡献。
- 不画骰子互联、共享灯路、长光束、实体骰飞行；不显示逐骰 `+N`、百分比、隐藏档位名或“暴击”文字。
- 游戏内最终信息只保留二元结果：`本关通过` 或 `本关未过`。总览中的五档标题仅为外部评审标注，不进入运行界面。

1. 先看同步视觉总览，判断五档是否形成连续且明确的塔体负载梯度。
2. 再看顺序原声总览，判断声音是否从卸载、锁定、双回弹、持续硬核到重击形成递进。
3. 若某一档需要调整，再打开对应单档原速文件，不用静态板替代动态判断。
