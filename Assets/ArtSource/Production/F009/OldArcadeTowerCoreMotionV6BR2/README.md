## 本版解决的问题

V6B-R1 只有一小段“贡献包入塔—过驱”，缺少一次出手的开端、六槽推进、回环因果和结果落位。本版重建为一条完整导演链：

- 从一次 `Space` 投骰开始，复用 V4R6“骰壳固定、骰面纵向滚轴、左到右停靠”。
- 结果稳定后自动进入结算，不保留主动结算，也没有加力窗口。
- 六个主槽从左到右连续结算；第二槽令第一槽再次触发时，第二槽主身份持续保持，第一槽只进入短促回声态。
- 每个产生即时分数的骰子都从自己的骰面中心凝出一包灯珠光晕，直接进入左侧积分塔；骰子之间没有关系线、共享灯路或实体位移。
- 强弱不靠逐骰 `+N`、百分比、评级名或暴击文字，而由包体核心、飞行余辉、塔内推进、承击和声音层数连续表达。
- 最后一颗重贡献把整手压力推入局部欠压、暖白击穿、余震回扫和 `StageClear` 结果驻留，MP4 不循环硬切。
