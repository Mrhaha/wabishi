# F009 V6B-R3 细节精修样片自审

审核日期：2026-07-22  
审核结论：`通过源区细节精修自审，待用户视觉验收`  
运行边界：仍为 `Assets/ArtSource/` 导演样片，未授权接入 Unity。

## 核心判断

V6B-R3 没有重新设计 V6B-R2，而是在已获用户认可的完整链路上补齐近看与运动中才能感知的机械细节。新增反馈从骰面析光开始，经过短尾积分包、塔口夹合、灯排承击记忆，最后收束到欠压灯丝、双触点和暴击余辉；主焦点仍稳定落在左侧积分塔。

## 本轮自审中保留的克制

1. 析光点固定为五枚，积分包短尾最多四枚；没有扩大成粒子群。
2. 塔口夹具保持为积分塔内部的小型器件，不延伸出管线或长光束。
3. 塔内玻璃反射只在承击与暴击窗口出现；全屏扫描层保持低透明度。
4. 磷光尘只在暴击局部短暂出现，共十二个确定性像素点，不使用持续火星雨。
5. 终端延迟回声不显示数值、评级或新的进度语义。
6. 仍以原速有声 MP4 作为动态主验收；两张抽帧板只定位节点，不代替连续性判断。

## 仍需用户判断的主观门槛

- 新增析光、短尾与塔口夹合是否让每次入塔更“有制作感”，同时没有变碎或变忙。
- 欠压残丝、双触点和灯排追亮是否让暴击更有层次，但仍像老式街机而不是科幻技能。
- 右侧弱波形、全屏低幅扫描和结果游标是否足够克制。
- R3 是否可冻结为暴击母版，并开始派生未达到、过关、超过、远超过四个收束。

用户确认前，本版只可作为动态方向候选，不能进入运行时资源生产或 Unity 接入。
