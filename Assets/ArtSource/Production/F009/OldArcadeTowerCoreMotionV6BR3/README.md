## R3 新增细节

1. **骰面析光**：积分包出现前，骰面内部先有五枚微型白炽灯丝点向中心收束；主触发较完整，回声更薄、更断续，不遮住稳定点数。
2. **积分包层次**：保留紧凑方形灯珠包，增加三到四枚衰减短尾和末段吸入加速。强弱继续由核心白热比例、尾迹寿命和塔体承击连续表达，不靠可计数弹丸。
3. **塔口机械吸收**：积分塔入口加入一组小型对开继电器夹口。光包接近时短开、命中时夹合，烟熏玻璃同时出现局部反射，不画长轨迹或能量管线。
4. **塔内压力记忆**：每次命中沿已点亮灯排形成一次很短的上行波纹，并留下缓慢爬行的双层扫描余辉；下一骰聚焦仍在上一击余辉中开始。
5. **目标锁扣**：真实越线后才执行一次约 `12°` 的机械扣合和小面积青白确认，青色不扩散成环境主色。
6. **暴击细化**：欠压阶段保留一根不稳定灯丝；释放时增加双触点吸合、两段硬闪、灯排追亮、少量局部磷光尘和衰减回扫。冲击仍只从积分塔发生，骰列不集体闪烁。
7. **终端与环境**：右侧点阵波形增加极弱的延迟回声，结果页保留一枚低亮游标；全屏只叠加非常轻的老 CRT 扫描呼吸，不改变 UI 语义色。
8. **临时声音**：在原连续机柜底噪上补入灯丝聚合、电磁夹口、目标双扣、暴击前触点和灯排余震点击；不是爆炸或魔法音色。
