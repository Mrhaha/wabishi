# 关间市场浅色实体键帽文字 V1

状态：人工退回、保留问题对照；纯色承载层破坏原键帽材质连续性，不得进入 Unity。后续修订见 `../MarketPhysicalKeyLabelsV2/`。

本目录只验证用户标注的五类浅色实体键帽：左侧卖出、三枚购买、右下离开市场。它们改用 `PhysicalKeyLabel` 实心深墨印刷字；暗色刷新键、商品名、六面、反馈条、Tooltip 和顶部资源窗继续使用已批准的 V4 LED 字。

## 单一推荐方案

- 字形来源：`Noto Sans SC` 可变字体，`600` 字重；本样张记录字体文件哈希。正式接入时必须把批准字形固化为仓库内字体或固体字图集，不依赖系统字体。
- 启用字色：`#180E08`，所有浅色实体键完全一致。
- 浅键底色：`#D6B88C`；字底对比度必须不低于 `7:1`。
- `1280×720`：购买 / 卖出使用 `22 px` 字号，离开市场使用 `24 px`；`1920×1080` 直接以 `33 / 36 px` 原生栅格化。
- 不使用点阵、辉光、描边、阴影、磨损遮罩或低透明细线。
- 悬停只亮键帽边缘；按下时键帽与文字共同下移；禁用态使用独立灰褐实心字并保持至少 `3:1`。
